## Generate your own

Every Bramble SDK is generated from our OpenAPI spec. You can generate a client in any language using [OpenAPI Generator](https://openapi-generator.tech):

```bash
# Download the spec
curl https://api.bramblehealth.com/v1/openapi.yaml -o bramble-openapi.yaml

# Generate a client (example: Go)
npx @openapitools/openapi-generator-cli generate \
  -i bramble-openapi.yaml \
  -g go \
  -o ./bramble-go-client
```
